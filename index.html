Vakken

Web

bij web leer je alles over het programmeren van web-applicaties. in het eerste leerjaar zul je onder andere leren hoe je een website zoals deze schrijft! in jaar 1 zul je leren werken met talen zoals HTML, CSS en PhP

Native

bij native leer je het schrijven van programma's die runnen op jouw device zelf. denk bij het eerste leerjaar aan simpele programma's. je zult leren werken met Python en een begin maken aan C#

Pra

bij pra wordt de theorie van de andere vakken toegepast. je krijgt hier opdrachten (vaak groepsopdrachten) die je kunt maken met de skills die je hebt geleerd tijdens web en Native
